# TTS 响度统一设计

## 目标

将离线 TTS 音频的平均响度统一到用户试听确认的 `-11 dBFS`，同时使用柔和限幅将峰值控制在 `-1 dBFS` 以内，避免直接削波造成明显失真。

## 方案

- 生成音频后计算整段波形的 RMS。
- 通过二分搜索求出达到 `-11 dBFS` 目标 RMS 所需的放大倍数。
- 使用 `ceiling * tanh(waveform * gain / ceiling)` 做柔和限幅，其中峰值上限为 `-1 dBFS`。
- 静音波形保持不变，避免无意义计算和数值异常。
- 将目标响度和峰值上限写入 `AudioProfile`；配置参与缓存键计算，因此旧音频缓存会自动失效并重新生成。

## 验收标准

- 普通 TTS 波形处理后的 RMS 为 `-11 dBFS`，允许浮点误差 `±0.05 dB`。
- 输出峰值不超过 `-1 dBFS`。
- 静音输入仍为静音。
- 缓存键随目标响度变化。
- 完整测试、代码检查、打包和冒烟启动均通过。
